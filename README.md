# AI Workplace Productivity Assistant

## 📌 Project Overview
AI Workplace Productivity Assistant is a modern, responsive web application designed to help professionals automate workplace tasks using AI.  
The platform provides a clean, SaaS-style dashboard with red-and-blue themed UI, enabling users to streamline communication, planning, and research in one place.

---

## ✨ Features
- **Smart Email Generator** – Create professional emails with structured prompts and editable outputs.  
- **Meeting Notes Summarizer** – Condense transcripts into key takeaways, decisions, and action items.  
- **AI Task Planner** – Convert natural language into structured tasks with deadlines
